ACME CORP — IT STANDARD OPERATING PROCEDURES
Version 3.2 | January 2025

=== GETTING IT SUPPORT ===

For all IT issues, use one of the following channels:

1. Self-Service Portal: it.acmecorp.internal (recommended for non-urgent issues)
2. Slack: #it-support channel — average response time 2 hours
3. Email: itsupport@acmecorp.com
4. Phone (urgent only): 555-IT-HELP (555-484-4357) — available Mon–Fri 8 AM–6 PM ET

Priority levels:
- P1 Critical: System down, cannot work. Response within 30 minutes.
- P2 High: Significant impact on work. Response within 2 hours.
- P3 Medium: Workaround available. Response within 1 business day.
- P4 Low: Minor issue or request. Response within 3 business days.

=== NEW EMPLOYEE SETUP ===

On your first day, IT will provision:
- MacBook Pro (standard) or Windows laptop (by request)
- Company email (@acmecorp.com)
- Slack access
- Zoom account
- Access to core tools (GitHub, Jira, Confluence, Figma depending on role)

To request additional software access:
1. Submit a Software Access Request in the IT portal
2. Obtain manager approval (manager will receive an email)
3. IT provisions access within 2 business days

=== PASSWORD & SECURITY POLICY ===

Passwords must be:
- At least 12 characters
- Mix of uppercase, lowercase, numbers, and symbols
- Changed every 90 days

Multi-Factor Authentication (MFA) is mandatory for all accounts. Use the Okta Verify app. Do not share passwords under any circumstances. If you suspect your account is compromised, call the IT emergency line immediately: 555-SEC-HELP.

VPN: Use GlobalProtect VPN when accessing internal resources from outside the office. Download from it.acmecorp.internal/vpn.

=== EQUIPMENT MANAGEMENT ===

Company equipment must not be used for personal business activities.

Laptop replacement cycle: 3 years for standard employees, 2 years for engineering roles.

If equipment is lost or stolen:
1. Report immediately to IT and your manager
2. IT will remotely wipe the device
3. File a police report if stolen
4. A replacement will be issued within 2 business days

Equipment must be returned upon departure. Unreturned equipment will be deducted from final paycheck at current replacement value.

=== SOFTWARE INSTALLATION ===

Approved software is available via the Managed Software Center on Mac or Software Center on Windows. You can install any approved software without IT involvement.

For software not in the catalog:
1. Submit a Software Request via the IT portal
2. IT Security reviews for compliance and licensing
3. Approval or denial within 5 business days
4. Do not install unapproved software — this may trigger a security alert

=== VIDEO CONFERENCING ===

Zoom is the standard video conferencing tool. All employees have a licensed Zoom account.

Best practices:
- Use a virtual background or blur if working from a personal space
- Mute when not speaking in meetings larger than 5 people
- Record meetings only with explicit permission from all participants
- External Zoom links should be password-protected

For large all-hands meetings (100+ attendees), contact it-events@acmecorp.com at least 1 week in advance.

=== DATA BACKUP ===

All work files should be stored in Google Drive or the company's designated cloud storage. Local-only files are not backed up.

Engineering code must be in the company GitHub organization (github.com/acmecorp). Do not use personal GitHub accounts for company code.

IT performs automated cloud backups daily. If you need to recover a deleted file from Google Drive, files are recoverable for up to 30 days. Contact IT for recovery requests.
